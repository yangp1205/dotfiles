# 这是一个保存个人配置文件的仓库

![Ubunut_Gnome](./pictures/ubuntu_gonme.png)

## vim
1. 安装插件管理工具vim-plug
2. 将vimrc文件替换
3. 安装插件
4. 安装cmake、python3.x-dev等工具
5. 进入目录安装YouCompleteMe

## zsh
1. 安装插件管理工具zplug
2. 替换zshrc文件
3. 安装插件和主题
4. 配置powerlevel10k（需要安装字体）
5. 安装oh-my-zsh
6. source .zshrc或者重启终端

## conky
* 使用wiki上的配置，修改为全透明背景以及显示位置

## ibus-rime
1. 使用明月简体拼音输入
2. 同时使用东风破管理插件（目前只有表情插件，而且还没有生效）
